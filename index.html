Doctype

<doctype> tag is used to inform the browser about the version of HTML used in the document. <!doctype html>

html

html means Hyper Text Markup Language. The html tag tells the browser that this is an HTML document.

head

When writing in HTML, the head tag is used to contain specific information about a web page, often referred to as metadata. The head tag is always used before the open body.

Body

The body is where all the website's content will be places, all text, images or links. All content must be within the body.

Title

The title tag is used to give the file a name, but only visible in the open file itself, for example the first or main file will always be called index, but when opened the title can be different. eg. <title>Your Title</title>

P

P, stands for paragraph. It is one of the most basic tags and used commonly to create paragraphs for websites. <p>paragraph</p>

span

The HTML span tag is used for grouping and applying styles to inline elements. <p> My cat has <span> style="color:black">black</span> fur.</p>

div

The div tag defines a division or a section in an HTML document. The div element is often used as a container for other HTML elements to style them with CSS or to perform certain tasks with JavaScript. <div style="background-color:coral">

Headings h1 to h6

Heading 1 <h1>heading 1</h1> is the biggest heading used for something important while heading 6 <h6>heading 6</h6> is used for less the least important headings

Formatting elements

<i>italics</i> changes text to italic.
<strong>Strong</strong> Changes text to bold.
<mark>text to be marked</mark> Marks text
<small>Small</small> Makes text small
<del>Text to be deleted</del> This doesnt delete text but rather puts a striketrough trough it.
<sub>Subscript</sub> Adds subscript effect to text.
<sup>Superscript</sup> Adds superscript to text.

Tables

A table is defined using the <table> element, and contains a number of table cells ( <td>, for "table data" ) which are organized into table rows ( <tr>). The markup for a table is always based on rows, never columns. Example:

<tr>
<th>First name</th>
<th>Last name</th> 
<th>Date of Birth</th>
</tr>
<tr>
<td>Jill</td>
<td>Smith</td> 
<td>20/03/1994</td>
</tr>
<tr>
<td>Eve</td>
<td>Jackson</td> 
<td>5/01/1995</td>
</tr>
</table>

Lists

An unordered list starts with the <ul> tag. Each list item starts with the <li> tag. The list items will be marked with bullets. Example:

  • First item
  • Second item
  • Third item
An ordered list starts with the <ol> tag. Each list item starts with the <li> tag. The list items will be marked with numbers by default
  1. First item
  2. Second item
  3. Third item

Images and Alt

To insert an image to html use <img src="image name" alt="Image name in case image doesnt show"> The purpose of "Alt" is to give the image a name if the image doesnt show for whatever reaason.

 

links

HTML links are hyperlinks. You can click on a link and jump to another document. When you move the mouse over a link, the mouse arrow will turn into a little hand. Example: Facebook

Quote and blockquote

<q> tag is used to insert a quatation. Example:

Facebook says: We will always be a free service

The <blockquote> tag specifies a section that is quoted from another source. Example:

Scientists analysing a meteorite have discovered the oldest material known to exist on Earth. They found dust grains within the space rock - which fell to Earth in the 1960s - that are as much as 7.5 billion years old.

Abbreviations

The <abbr> tag defines an abbreviation or an acronym, like "HTML", "Mr.", "Dec.", "ASAP", "ATM". Example: The NASA was founded in July 29, 1958.

Meta

Metadata is data (information) about data. The <meta> tag provides metadata about the HTML document. Metadata will not be displayed on the page. Meta elements are typically used to specify page description, keywords, author of the document, last modified, and other metadata. Example: <meta name="keywords" content="HTML, CSS, XML, XHTML, JavaScript">

Special symbols and character entities

To add symbols to an HTML page, you can use an HTML entity name. If no entity name exists, you can use an entity number, a decimal, or hexadecimal reference. Examples:
&copy; will show = ©
&reg; will show = ®
&amp; will show = &
&euro; will show = €
&nbsp; will show the non-breaking space=  
&lt; will show = <
&gt; will show = >
